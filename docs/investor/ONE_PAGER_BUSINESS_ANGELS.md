# DANIELA by TierCore — One Pager para Business Angels

## Qué es
DANIELA es un Infrastructure Kernel para hoteles y resorts de lujo que convierte datos técnicos dispersos en decisiones operativas accionables usando IA.

## Problema
Las propiedades premium operan con BMS fragmentados por protocolo/fabricante, con detección tardía de fugas, baja visibilidad del rendimiento HVAC y decisiones reactivas que aumentan OPEX y riesgo operacional.

## Solución
- Capa inteligente agnóstica al fabricante
- Detección de fugas por patrón nocturno contextual
- Optimización de chillers con estimación de ahorro diario
- Arquitectura modular para escalar protocolos y analítica

## Estado actual (Feb 2026)
- Prototipo funcional v0.2
- 28 archivos Python y ~3.4k LOC
- 2 módulos IA operativos (Leak Detector, Chiller Optimizer)
- 2 protocolos base (BACnet/IP, Modbus)
- 3 suites de tests
- Simulación de 191 viviendas y equipos críticos

## Valor para cliente
- Menor costo energético y de agua
- Menos operación reactiva y mejor priorización técnica
- Mayor continuidad de servicio en hospitalidad de lujo
- Base de datos operativa para escalar mantenimiento predictivo
